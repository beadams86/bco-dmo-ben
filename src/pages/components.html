Overview

This BCO-DMO website template uses the lightweight, open source CSS framework Bulma for baseline elements such as buttons, forms, layout grid, and more.

There's a few reasons why we currently use Bulma, but the biggest benefits are:

  1. A well tested, mobile-first, responsive framework
  2. Customizable and Modular; our version of Bulma is custom to BCO-DMO
  3. A large library of standardized, but customized UI components and elements
  4. ...and it's all well documented!

.has-text-justified Makes the text justified
